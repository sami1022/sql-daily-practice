## Day 72 – Recursive CTE (Hierarchical Queries)

### Problem Statement

You are given a table **employees**:

- employee_id (INT)
- employee_name (VARCHAR)
- manager_id (INT)

This table represents an organizational hierarchy.

Write SQL queries to:

1. Retrieve the full employee hierarchy
2. Find all employees under a specific manager
3. Display hierarchy levels
4. Trace management chain
5. Understand recursive query structure

Focus on using Recursive CTEs for hierarchical data.
