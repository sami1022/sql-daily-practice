## Day 77 – ROLLBACK (Error Handling)

### Problem Statement

You are given a table **accounts**:

- account_id (INT)
- account_name (VARCHAR)
- balance (DECIMAL)

Write SQL queries to:

1. Start a transaction and perform updates
2. Rollback the transaction if an error occurs
3. Demonstrate safe transaction handling
4. Compare COMMIT vs ROLLBACK
5. Understand failure scenarios in transactions

Focus on undoing changes using ROLLBACK.
