## Day 61 – Set Operations (Mixed Practice)

### Problem Statement

You are given three tables:

**online_customers**
- customer_id (INT)
- name (VARCHAR)

**store_customers**
- customer_id (INT)
- name (VARCHAR)

**mobile_customers**
- customer_id (INT)
- name (VARCHAR)

Write SQL queries to:

1. Combine all customers from all channels
2. Find customers common in online and store
3. Find customers present only in mobile
4. Count total unique customers
5. Combine results and label source

Focus on mixing set operations effectively.
