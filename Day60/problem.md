## Day 60 – EXCEPT (Difference Between Sets)

### Problem Statement

You are given two tables:

**online_customers**
- customer_id (INT)
- name (VARCHAR)

**store_customers**
- customer_id (INT)
- name (VARCHAR)

Write SQL queries to:

1. Find customers present in online but not in store
2. Reverse the comparison (store but not online)
3. Compare EXCEPT with LEFT JOIN
4. Count unmatched records
5. Understand duplicate behavior

Focus on retrieving difference between result sets using EXCEPT.
