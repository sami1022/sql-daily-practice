## Day 76 – Transactions (BEGIN, COMMIT)

### Problem Statement

You are given a table **accounts**:

- account_id (INT)
- account_name (VARCHAR)
- balance (DECIMAL)

Write SQL queries to:

1. Transfer money from one account to another using transaction
2. Update multiple records safely using BEGIN and COMMIT
3. Ensure data consistency during updates
4. Understand transaction control flow
5. Demonstrate atomic operations

Focus on using transactions for safe data operations.
