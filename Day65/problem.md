## Day 65 – Ranking Functions (RANK & DENSE_RANK)

### Problem Statement

You are given a table **sales**:

- sale_id (INT)
- employee_id (INT)
- region (VARCHAR)
- amount (DECIMAL)
- sale_date (DATE)

Write SQL queries to:

1. Rank sales by amount using RANK
2. Rank sales by amount using DENSE_RANK
3. Rank sales within each region
4. Find top 3 sales in each region
5. Compare RANK and DENSE_RANK results

Focus on ranking functions in SQL.
