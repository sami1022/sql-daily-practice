## Day 82 – Stored Procedures (Basics)

### Problem Statement

You are given a table **accounts**:

- account_id (INT)
- account_name (VARCHAR)
- balance (DECIMAL)

Write SQL to:

1. Create a stored procedure to fetch account details
2. Create a procedure to deposit money
3. Create a procedure to withdraw money
4. Use parameters in stored procedures
5. Call stored procedures

Focus on basic stored procedure usage.
