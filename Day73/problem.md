## Day 73 – CTE vs Subquery (Comparison)

### Problem Statement

You are given a table **orders**:

- order_id (INT)
- customer_id (INT)
- amount (DECIMAL)
- order_date (DATE)

Write SQL queries to:

1. Calculate total spending per customer using a subquery
2. Calculate total spending per customer using a CTE
3. Find customers whose spending is above average using subquery
4. Solve the same problem using a CTE
5. Compare readability and structure of both approaches

Focus on understanding when to use CTE vs subquery.
