## Day 69 – Practice (Analytics SQL Problems)

### Problem Statement

You are given a table **orders**:

- order_id (INT)
- customer_id (INT)
- amount (DECIMAL)
- order_date (DATE)

Write SQL queries to:

1. Find each customer's total spending
2. Rank customers by total spending
3. Find the highest order per customer
4. Calculate running total of customer spending
5. Find the previous order amount for each customer

Focus on solving analytics problems using window functions.
