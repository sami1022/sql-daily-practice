### Important Patterns

Self Join:
- Used when comparing rows in the same table
- Example: employee vs manager

GROUP BY + HAVING:
- Used for filtering aggregated results
- Example: customers with multiple orders

Gap Detection:
- Used to find missing values
- Important in real-world datasets
