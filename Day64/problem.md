## Day 64 – Window Functions (ROW_NUMBER)

### Problem Statement

You are given a table **sales**:

- sale_id (INT)
- employee_id (INT)
- region (VARCHAR)
- amount (DECIMAL)
- sale_date (DATE)

Write SQL queries to:

1. Assign row numbers to all sales records
2. Rank sales within each region
3. Retrieve the top sale per region
4. Identify the latest sale for each employee
5. Use ROW_NUMBER with ORDER BY

Focus on understanding ROW_NUMBER and window functions.
