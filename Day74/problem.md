## Day 74 – Practice (CTE Problems)

### Problem Statement

You are given a table **orders**:

- order_id (INT)
- customer_id (INT)
- amount (DECIMAL)
- order_date (DATE)

Write SQL queries to:

1. Calculate total spending per customer using a CTE
2. Find the top 3 customers by spending
3. Identify customers whose spending is above the overall average
4. Calculate cumulative spending per customer using CTE + window function
5. Use multiple CTEs to simplify a complex query

Focus on building multi-step analytical queries using CTEs.
