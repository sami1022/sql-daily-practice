## Day 87 – SQL for Analytics (KPIs & Metrics)

### Problem Statement

You are given a table **orders**:

- order_id (INT)
- customer_id (INT)
- amount (DECIMAL)
- order_date (DATE)

Write SQL queries to:

1. Calculate total revenue
2. Calculate average order value (AOV)
3. Count total number of orders
4. Calculate revenue per customer
5. Find daily revenue trends

Focus on business metrics and KPI calculations.
