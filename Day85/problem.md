## Day 85 – Practice (Advanced SQL)

### Problem Statement

You are given a table **accounts**:

- account_id (INT)
- account_name (VARCHAR)
- balance (DECIMAL)

And a table **transactions**:

- transaction_id (INT)
- account_id (INT)
- amount (DECIMAL)
- transaction_date (DATE)

Write SQL to:

1. Create a stored procedure for money transfer
2. Create a function to check sufficient balance
3. Use trigger to log transactions automatically
4. Combine procedure + function for validation
5. Ensure safe and automated operations

Focus on combining advanced SQL concepts.
