## Day 66 – LEAD & LAG (Row Comparison)

### Problem Statement

You are given a table **sales**:

- sale_id (INT)
- employee_id (INT)
- amount (DECIMAL)
- sale_date (DATE)

Write SQL queries to:

1. Retrieve the previous sale amount for each employee
2. Retrieve the next sale amount
3. Calculate difference between current and previous sale
4. Compare consecutive sales
5. Analyze sales trend using LAG

Focus on row comparison using LEAD and LAG window functions.
