## Day 90 – Product Metrics (Data Analyst Focus)

### Problem Statement

You are given a table **user_activity**:

- user_id (INT)
- activity_date (DATE)
- activity_type (VARCHAR)  -- login, purchase

Write SQL queries to:

1. Calculate Daily Active Users (DAU)
2. Calculate Monthly Active Users (MAU)
3. Calculate churned users (users who stopped returning)
4. Calculate retention rate
5. Identify most active users

Focus on product metrics and user engagement.
