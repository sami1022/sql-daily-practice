## Day 97 – SQL Optimization (Performance & Tuning)

### Problem Statement

You are given large tables:

**orders**
- order_id (INT)
- customer_id (INT)
- product_id (INT)
- amount (DECIMAL)
- order_date (DATE)

**customers**
- customer_id (INT)
- name (VARCHAR)

---

### Tasks

1. Optimize a query to find top customers by revenue
2. Improve performance of JOIN queries
3. Use indexing effectively
4. Avoid full table scans
5. Analyze query execution using EXPLAIN

Focus on writing efficient SQL queries.
