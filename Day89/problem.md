## Day 89 – User Analysis (Funnels & Retention)

### Problem Statement

You are given a table **user_events**:

- user_id (INT)
- event_name (VARCHAR)  -- signup, login, purchase
- event_date (DATE)

Write SQL queries to:

1. Count number of users at each funnel stage
2. Calculate conversion rate from signup to purchase
3. Find daily active users (DAU)
4. Identify users who returned after signup
5. Calculate retention rate

Focus on user behavior and funnel analysis.
