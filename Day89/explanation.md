## Day 89 – User Analysis (Conceptual Explanation)

User analysis focuses on
understanding how users
interact with a product.

It helps businesses
improve growth and retention.

---

### Funnel Analysis

A funnel represents
user journey stages.

Example funnel:

Signup → Login → Purchase

It helps measure:
- drop-offs
- conversion rates

---

### Conversion Rate

Conversion rate measures
how many users move
from one stage to another.

Example:
Signup → Purchase

---

### Active Users

Active users measure engagement.

Types:
- DAU (Daily Active Users)
- MAU (Monthly Active Users)

---

### Retention Analysis

Retention measures
how many users return
after initial interaction.

High retention = strong product

---

### Behavioral Insights

User analysis helps answer:

- Are users returning?
- Where do users drop off?
- Which features are used most?

---

### Real-World Applications

Used in:

- mobile apps
- SaaS platforms
- e-commerce
- social media products

---

### Interview Importance

Common questions:

- Calculate conversion rate
- Find retention rate
- Analyze funnels
- Compute DAU/MAU

---

### Summary

User analysis includes:

- funnels
- conversion rates
- retention
- engagement metrics

This is core product analytics SQL.
