## Day 80 – Practice (Transactions & Locks)

### Problem Statement

You are given a table **accounts**:

- account_id (INT)
- account_name (VARCHAR)
- balance (DECIMAL)

Write SQL queries to:

1. Transfer money between accounts safely
2. Use ROLLBACK when an error occurs
3. Demonstrate ACID properties in action
4. Apply row-level locking using FOR UPDATE
5. Prevent inconsistent updates in concurrent transactions

Focus on safe and reliable transaction handling.
