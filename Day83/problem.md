## Day 83 – Functions (User-Defined Functions)

### Problem Statement

You are given a table **accounts**:

- account_id (INT)
- account_name (VARCHAR)
- balance (DECIMAL)

Write SQL functions to:

1. Create a function to return account balance by ID
2. Create a function to calculate total balance of all accounts
3. Create a function to check if balance is sufficient
4. Use parameters in functions
5. Call functions in SELECT queries

Focus on returning values using SQL functions.
