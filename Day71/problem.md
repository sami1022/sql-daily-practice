## Day 71 – CTE (WITH Clause)

### Problem Statement

You are given a table **orders**:

- order_id (INT)
- customer_id (INT)
- amount (DECIMAL)
- order_date (DATE)

Write SQL queries to:

1. Calculate total spending per customer using a CTE
2. Find customers whose spending is above average
3. Rank customers by total spending using a CTE
4. Use multiple CTEs in a single query
5. Improve query readability using CTE

Focus on using the WITH clause effectively.
